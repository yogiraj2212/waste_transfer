# Waste Segregation AI

A Deep Learning based web application that classifies waste into:

- Plastic
- Metal
- Paper

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS

## Features
- Upload waste image
- Predict waste category
- User-friendly web interface

## Project Structure

├── dataset/
├── templates/
│ └── index.html
├── app.py
├── train.py
└── model.h5

## Run Locally

pip install -r requirements.txt

python app.py
